# 🏥 Machine Learning Model Building - Healthcare Dataset

## 📌 Project Overview
This project focuses on building a machine learning model using preprocessed healthcare data. The aim is to predict whether a patient has a disease based on health-related features.

---

## 🎯 Objectives
- Load preprocessed dataset
- Perform train-test split
- Train a machine learning model
- Make predictions
- Evaluate model performance

---

## 📊 Dataset
- Processed healthcare dataset from preprocessing project
- Includes encoded and scaled features

---

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn

---

## 🔄 Workflow
Load Data → Train-Test Split → Model Training → Prediction → Evaluation

---

## 🤖 Model Used
- Logistic Regression

---

## 📈 Result
The model achieved high accuracy on the dataset.

---

## ⚠️ Note
Preprocessing was performed before train-test split, which may lead to data leakage. In real-world scenarios, preprocessing should be applied after splitting the dataset.
